## 🚀 Featured Engineering

### 🏦 [Amanat — PSX Shariah Portfolio Tracker](https://amanat-psx.vercel.app)
> **Django DRF · React · PostgreSQL · Celery · Redis · Recharts**

A full-stack, production-deployed SaaS platform for tracking Shariah-compliant stock portfolios on the Pakistan Stock Exchange.

**Key Engineering Decisions:**
- Async background tasks via **Celery + Redis** for automated daily price updates — no manual intervention needed
- Decoupled **REST API** (Django DRF) with a standalone **React SPA** frontend
- Custom Zakat/tax routing logic tailored to Islamic finance rules
- Interactive data visualization with **Recharts** for portfolio performance dashboards

🔗 [Live App](https://amanat-psx.vercel.app) · [Frontend Repo](https://github.com/RoadtoFire/psx-frontend) · [Backend Repo](https://github.com/RoadtoFire/psx-api)
